# Milque.js
A declarative, rapid game prototyping framework for the web.

Inspired by [Milky.js](http://www.lessmilk.com/milkyJS/).


- WYSIWYG Pixel-Perfect Image Editor
- Entity Editor
    - JSON file
    - Entity creation
- Scene Editor
    - Resource list
    - Engine list
    - Entity setup
    - JSON file
- Render Editor
    - JavaScript file?
